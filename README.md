# go-omf

Go library for reading and writing OMF (Open Mining Format) files.

## Supported elements

- PointSet — point cloud data
- PolyLine — polyline data
- TriSurf — triangle mesh surfaces
- TetraMesh — tetrahedral volume meshes
- Volume — 3D regular grid volumes

## Features

- Element management with type-based API
- Bounding box calculation
- Binary serialization
